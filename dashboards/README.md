# Dashboards

Add BI dashboards built from the marts layer (e.g., `mart_daily_metrics`, `mart_vendor_metrics`).

## Screenshot instructions
1. Build your dashboard in QuickSight, Tableau, or Power BI.
2. Export a screenshot to `dashboards/screenshots/`.
3. Update this README with the embedded image.

Suggested tiles:
- Daily trips and revenue trend.
- Vendor performance bar chart.
